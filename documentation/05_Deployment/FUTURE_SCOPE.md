# Future Scope

## Potential Enhancements
- Driver emotion recognition and mood analysis
- Lane detection and road context awareness
- Vehicle CAN bus integration for speed and steering data
- WebSocket or MQTT telemetry streaming
- Cloud deployment with autoscaling and monitoring
- Mobile companion app for alerts and profile management
- Better fatigue estimation using additional sensors
- Advanced AI models for multi-modal detection
- Multi-driver and occupant monitoring
- Fleet management analytics and dashboards

## Backend Improvements
- Add GPU acceleration support for YOLO and InsightFace
- Replace SQLite with a scalable database for fleets
- Add authentication and authorization for API access
- Migrate to a service-based detection pipeline
- Add a history and logging database for telemetry events

## Research Directions
- Add emotion and stress detection
- Use radar or infrared for low-light reliability
- Integrate expert system rules with machine learning
- Support predictive maintenance and risk forecasting
